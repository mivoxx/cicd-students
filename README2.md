## Install Git on Linux
This guide shows how to install *[Git](https://git-scm.com/)* from *[apt](https://wiki.debian.org/Apt)* on **Debian/Ubuntu**.

## Step 1: Update the apt package information index
```bash
sudo apt update
```
## Step 2: Install Git using apt
```bash
sudo apt install git
```
## step 3: Verify the installation was successful by typing `git --version`:
```bash
git --version
```
